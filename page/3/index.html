PS制作人物与纸张背景的穿插效果；2019.01.08

1、首先在PS新建一个文件 [Ctrl+N新建图形文件]，大小设置是（宽度1024、高度649），填充白颜色[Shift+F5填充]；接着点击选择：图层 >图层样式 >渐变叠加（混合模式：叠加，不透明度：19%，白色到黑色渐变：20%黑，样式：径向，与图层对齐打勾，角度：90度，缩放：150%）；再点击选择：图层 >图层样式 >渐变叠加（混合模式：正常，不透明度：20%，白色到黑色渐变：50%黑，样式：径向，与图层对齐打勾，角度：90度，缩放：150%）

2、打开人物素材，使用钢笔 [P钢笔工具]抠取下来，然后拖入画布背景图层，调整尺寸、大小

3、新建一个图层（形状1） [Shift+Ctrl+N新建图层]，使用钢笔 [P钢笔工具]画一个斜横着细长条状的选区（穿过背部），选择选区更改 [Ctrl+Enter选区更改]选中之后为虚线，填充黑颜色[Shift+F5填充]，此图层放在人物图层的下面

4、新建一个图层（形状2） [Shift+Ctrl+N新建图层]，使用钢笔 [P钢笔工具]画一个斜竖着细长条状的选区（手肘位置），选择选区更改 [Ctrl+Enter选区更改]选中之后为虚线，填充黑颜色[Shift+F5填充]，此图层放在人物图层的下面

5、按住Ctrl键并鼠标点击形状1的图层缩览图，找出形状1的选区；使用套索[L套索工具]画取人物上半部分，再点击人物的图层缩览图，选择点击添加图层蒙版，就可将人物下半部分消除

6、右键点击人物图层的图层蒙板，选择停用图层蒙板；接着按住Ctrl键并鼠标点击形状2的图层缩览图，找出形状2的选区，使用套索[L套索工具]画取人物手臂部分；再右键点击人物图层的图层蒙板，选择启用图层蒙板，然后在手臂部分的选区填充白颜色[Shift+F5填充]，就可将手臂显露出来

7、点击人物的图层缩览图，接着点击选择：图层 >图层样式 >投影（混合模式：正片叠底，黑颜色，不透明度：30%，角度：1度，使用全局光打勾，距离：10像素，扩展：0%，大小：20像素，等高线：线性，图层挖空投影打勾），就可做出投影效果

7、最后，合并 [Ctrl+Shift+E合并可见图层]，保存[Ctrl+S保存当前图像]

PS制作合成透明鸡蛋；2019.01.07

1、先将鸡蛋背景图层拖进PS，选择复制图层 [Ctrl+J复制拷贝图层]，使用仿制图章[S仿制图章、图案图章工具]把黑色鸡蛋涂抹掉；再用修补 [J修补工具]把边缘明显的痕迹去掉

2、打开气泡素材，使用椭圆选框[M矩形、椭圆选框工具]将气泡选取抠下来，然后拖入鸡蛋背景图层，调整尺寸、大小替换原来黑鸡蛋的位置，选择去色 [Ctrl+Shift+U去色]，把不透明度降低一些（不透明度：30%），还要选择点击添加图层蒙版，用黑色画笔 [B画笔工具]将里面多余部分涂抹掉

3、将蛋黄图层拖进置入PS [Alt+F+L置入]，调整尺寸、大小使蛋黄在鸡蛋里面，选择点击添加图层蒙版，用黑色画笔 [B画笔工具]将里面多余部分涂抹掉，还要在混合更改{正常}栏框下点击选择正片叠底

4、使用渐变 [G渐变、油漆桶工具]制作一个粉色到黄色的渐变背景，在混合更改{正常}栏框下点击选择柔光，使其色彩更有氛围

5、最后，合并 [Ctrl+Shift+E合并可见图层]，保存[Ctrl+S保存当前图像]

PS更换证件照底色；2019.01.06

1、先将人物证件照的背景图层拖进PS

2、使用钢笔 [P钢笔工具]，将人物的整体轮廓锚点出来；整个人物锚点完成之后，要形成一个闭合的空间，即起点和终点要重合；选择选区更改 [Ctrl+Enter选区更改]，选中之后为虚线

3、新建一个图层 [Shift+Ctrl+N新建图层]；接着点击背景图层，复制人物 [Ctrl+C拷贝选取的图像或路径]；然后点击新建的图层，粘贴人物 [Ctrl+V将剪切板的内容粘贴到当前图形中]

4、点击背景图层；将原有图层替换成白底 [Ctrl+Backspace用背景色填充所选区域或整个图层]；或者可以直接填充颜色 [Shift+F5填充]，比如黄色

5、最后，合并 [Ctrl+Shift+E合并可见图层]，保存[Ctrl+S保存当前图像]

PS制作炫酷的冰火两重天火焰特效；2019.01.05

1、先将人物背景图层拖进PS，再选择黑色背景的火焰图层拖进置入PS [Alt+F+L置入]，把火焰图层在混合更改{正常}栏框下点击选择滤色，然后调整尺寸、大小，（因为人物背景是黑色的，所以再选择黑色背景的火焰图层，就不需要抠取火焰，直接选择滤色的混合更改就可以了）

2、火焰从人物或物体的边缘开始放置，不断反复的添加火焰，直到火焰的数量另您满意为止；给火焰图层添加一个图层蒙版，让它不会出现在里面的胳膊上；可以在火焰图层使用橡皮擦工具 [E橡皮擦工具]结合将不透明度调小，把多余的地方进行涂抹消除；或者在蒙板图层使用画笔 [B画笔工具]（黑颜色），把火焰边缘的地方进行涂抹修饰

3、所有的火焰图层的混合更改都设为滤色；完成后，把所有的火焰图层点击选择创建新组，命名为“fire”

4、在“fire”图层组的下面新建一个图层[Ctrl+Shift+N 新建图层]，命名为“fire background”，使用圆点硬画笔 [B画笔工具]（颜色#8b2c0d），在火焰背后涂抹

5、在“fire background”图层的上面新建一个图层[Ctrl+Shift+N 新建图层]，新建的图层1在混合更改{正常}栏框下点击选择柔光，使用圆点硬画笔 [B画笔工具]（颜色#8b2c0d），在火焰上随便画画

6、接着再在上面新建一个图层，再新建的图层2在混合更改{正常}栏框下同样点击选择柔光，使用圆点硬画笔 [B画笔工具]（颜色#dd4e05），在火焰和人物的上画一个巨大的橙色光圈，让所有的光线呈现橙色，照射到人物的左脸上，把不透明度降低一些（不透明度：70%）

7、再次新建一个图层，再新建的图层3在混合更改{正常}栏框下同样点击选择柔光，把不透明度降低一些（不透明度：70%），使用圆点硬画笔 [B画笔工具]（颜色#ff894f），画一个大一点的橙色光晕；左键双击图层3的缩览图就可打开“图层样式”面板（混合颜色带：灰色；鼠标拖动“下一图层”调整使其达到最佳效果）；完成后，把刚刚所有的画笔涂画图层点击选择创建新组，命名为“Fire Fill”

8、在“Fire Fill”图层组的上面新建一个图层[Ctrl+Shift+N 新建图层]，使用圆点硬画笔 [B画笔工具]（颜色#9e0000），涂抹在整个图层的右侧，把不透明度降低一些（不透明度：27%）

9、再次新建一个图层，在混合更改{正常}栏框下同样点击选择颜色，使用圆点硬画笔 [B画笔工具]（颜色#0e28df），涂抹在整个图层的左侧，把不透明度降低一些（不透明度：27%）

10、点击选择{调整}下面的“色彩平衡”调整图层（青色+15, 洋红-20，黄色-9）

11、点击选择{调整}下面的“曲线”调整图层（预设：自定，颜色：红，拖动红标线，使火焰更加真实），选择点击添加图层蒙版，用画笔 [B画笔工具]（黑颜色）涂抹没有火焰的地方

12、点击选择{调整}下面的“曲线”调整图层（预设：自定，颜色：RGB，拖动标线，使图像变暗一点），选择点击添加图层蒙版，用画笔 [B画笔工具]（黑颜色）涂抹火焰的地方和火焰周围

13、最后，合并 [Ctrl+Shift+E合并可见图层]，保存[Ctrl+S保存当前图像]

PS使用通道抠图快速抠出书画中的梅花；2019.01.04

1、先将背景图层拖进PS，选择复制图层 [Ctrl+J复制拷贝图层]

2、在通道面板RGB模式下观察三个通道，分别是红绿蓝；（找一个黑白对比度大的）比如蓝通道，右键点击蓝通道选择“复制通道”就可得到蓝 拷贝通道图层

3、接着，把其它通道图层的眼睛关闭，只开启蓝 拷贝通道图层的眼睛；选择调整色阶[Ctrl+L调整色阶]（调整色阶的目标就是把要扣的区域变成黑色的，还不能有其它杂色）

4、按住“Ctrl”键并同时左键点击蓝 拷贝通道图层；然后回到图层面板，左键点击图层1，选择反选 [Ctrl+Shift+I 反向选择]，再复制图层 [Ctrl+J复制拷贝图层]，即可完成。

PS使用通道制作下雨GIF动画；2018.12.29

1、先将背景图层拖进PS

2、选择：窗口 >动作[Alt+F9打开窗口的动作]，接着点击选择：创建新组（名称：默认动作1），再点击选择：创建新动作（名称：下雨，在默认动作1组下，功能键：F2）；点击选择“记录”即可开始录制

3、打开通道面板，点击选择“创建新通道”就可得到Alpha 1，设置前景色为白色、背景色为黑色 [D复位色板]

4、点击选择：滤镜 >像素化 >点状化（单元格大小：4；如果雨丝要细相应的数据就设置小一点）

5、选择：图像 >调整 >色阶 [Ctrl+L调整色阶]（预设：自定，通道：Alpha 1，第一框：47、第二框：9.99第三框：75；这一步可以设置色阶中的参数得到分布均匀的小点）

6、点击选择：滤镜 >模糊 >动感模糊（角度：90度，距离：25像素）

7、选择当前通道全选 [Ctrl+A当前通道全选]，接着复制 [Ctrl+C复制]，点击一下“RGB”通道；然后回到图层面板，新建一个图层 [Shift+Ctrl+N新建图层]，再粘贴 [Ctrl+V粘贴]，还要在混合更改{正常}栏框下点击选择“滤色”

8、选择：窗口 >动作[Alt+F9打开窗口的动作]，之前制作的步骤已经全部被录制了，接着点击选择：停止播放/记录，再按两次设置好的快捷键“F2”系统就会重复之前的工作，复制两个“图层2”“图层3”

9、点击选择：窗口 >时间轴,点击切换选择为“创建帧动画”，而不是“创建视频时间轴”

10、在{创建帧动画}模式下就会出现第一帧，点击选择第一帧，将时间设置为0.2秒，要点击开启“背景图层”和“图层1”的眼睛，把其它图层的眼睛关闭

11、接着点击选择“复制所选帧”，就会出现第二帧，点击选择第二帧，将时间设置为0.2秒，要点击开启“背景图层”和“图层2”的眼睛，把其它图层的眼睛关闭

12、再点击选择“复制所选帧”，就会出现第三帧，点击选择第三帧，将时间设置为0.2秒，要点击开启“背景图层”和“图层3”的眼睛，把其它图层的眼睛关闭

注：第一帧 = 图层1，第二帧 = 图层2，第三帧 = 图层3；背景图层的眼睛永远一直开启，不要关闭

13、点击选择：选择循环选项（设为“永远”）

14、点击选择：播放动画，先预览一下效果；没问题的话，接着存储为 Web 所用格式… [Alt+Shift+Ctrl+S存储为 Web 所用格式…]即可完成。

PS使用滤镜制作下雨GIF动画；2018.12.29

1、先将背景图层拖进PS

2、选择：窗口 >动作[Alt+F9打开窗口的动作]，接着点击选择：创建新组（名称：默认动作1），再点击选择：创建新动作（名称：下雨，在默认动作1组下，功能键：F2）；点击选择“记录”即可开始录制

4、完成以后图层会变透明，形成下雨的效果；但是边缘的颜色会有些深色、杂色，把图层调整大一些 [Ctrl+T自由变换]就可以隐藏边缘

5、选择：窗口 >动作[Alt+F9打开窗口的动作]，之前制作的步骤已经全部被录制了，接着点击选择：停止播放/记录，再按两次设置好的快捷键“F2”系统就会重复之前的工作，复制两个“图层2”“图层3”

6、点击选择：窗口 >时间轴,点击切换选择为“创建帧动画”，而不是“创建视频时间轴”

7、在{创建帧动画}模式下就会出现第一帧，点击选择第一帧，将时间设置为0.1秒，要点击开启“背景图层”和“图层1”的眼睛，把其它图层的眼睛关闭
8、接着点击选择“复制所选帧”，就会出现第二帧，点击选择第二帧，将时间设置为0.1秒，要点击开启“背景图层”和“图层2”的眼睛，把其它图层的眼睛关闭

9、再点击选择“复制所选帧”，就会出现第三帧，点击选择第三帧，将时间设置为0.1秒，要点击开启“背景图层”和“图层3”的眼睛，把其它图层的眼睛关闭

注：第一帧 = 图层1，第二帧 = 图层2，第三帧 = 图层3；背景图层的眼睛永远一直开启，不要关闭

10、点击选择：选择循环选项（设为“永远”）

11、点击选择：播放动画，先预览一下效果；没问题的话，接着存储为 Web 所用格式… [Alt+Shift+Ctrl+S存储为 Web 所用格式…]即可完成。

PS制作雪景；2018.12.28

1、先将人物背景图层拖进PS

2、点击选择{调整}下面的：通道混和器（先把“单色”打勾，预设：自定，输出通道：灰色，使照片成为黑白效果；再调整：红色-10%、绿色+168%、蓝色-60%，使远处的草地也呈现白色：）；还要在混合更改{正常}栏框下点击选择“变亮”，减弱对人物的影响

3、点击选择{调整}下面的：可选颜色（预设：自定，颜色：红色，调整：黑色-78%，对人物的皮肤和小桥的颜色进行调修；再选择颜色：中性色：调整：青色-10%、洋红-5%、黄色-7%、黑色-10%，使画面整体色调变亮）

4、使用椭圆选框 [M矩形、椭圆选框工具]（羽化：50像素，模式：正常）在人物四周围创建一个选区，选择反选 [Ctrl+Shift+I 反向选择]，点击选择{调整}下面的：色相/饱和度（明度+91，使图像周围调亮）

5、点击选择{调整}下面的：亮度/对比度（亮度：25、对比度：12，使画面适当调亮并增加对比度）；但调亮画面后，婚纱的亮度都失去了层次，就可使用画笔 [B画笔工具]选择黑色，在婚纱部分涂抹，使调整图层时不影响婚纱部分的区域

6、将背景图层复制一个[Ctrl+J复制拷贝图层]，再把刚刚复制好的“背景 拷贝”图层拖拽至最上面顶层

7、接着对“背景 拷贝”图层设置前景色为黑色、背景色为白色，点击选择：滤镜 >滤镜库 >素描 >影印（细节：1、暗度：6，使图像变成线描效果）；再在混合更改{正常}栏框下点击选择“颜色加深”，使画面中保留线描，体现手绘风格；还要选择点击添加图层蒙版，使用画笔 [B画笔工具]选择黑色，在人物身体、婚纱边缘涂抹，就可将多余的线条隐藏

8、打开通道面板，点击选择“创建新通道”就可得到Alpha 1，设置前景色为白色、背景色为黑色；接着点击选择：滤镜 >像素化 >点状化（单元格大小：5，便生成灰色杂点）；再点击选择：图像 >调整 >阈值（阈值色阶：41，使灰色杂点变得清晰）；点击选择“将通道作为选区载入”，就可载入通道中的选区

9、回到图层面板，新建一个图层 [Shift+Ctrl+N新建图层]，在选区内填充白颜色 [Shift+F5填充]，接着取消选择 [Ctrl+D取消选择]；再点击选择:滤镜 >模糊 >动感模糊（角度：60度，距离：5像素，便可对杂点进行模糊，制作出雪花飘落效果）；还要选择点击添加图层蒙版，使用画笔 [B画笔工具]选择（1、颜色：深灰色，大小：50像素，不透明度：25%，将人物身体上的雪花适当隐藏、变得透明；2、颜色：黑色，大小：5像素，不透明度：100%，将人物脸上的雪花涂抹擦除）

10、将花朵图层抠取拖入主背景，调整好尺寸、比例，使花朵戴在人物头上

11、最后，合并 [Ctrl+Shift+E合并可见图层]，保存[Shift+Ctrl+S保存当前图像]

PS高反差保留磨皮设计技巧；2018.12.27

这个磨皮方法的关键是先复制一个通道，然后用高反差把反差大的地方弄出来，然后用计算把这些反差大的地方增强，再做出选区，用曲线调一下就能达到磨皮的目的！

1、先将人物背景图层拖进PS，点击选择：通道 >蓝色通道，复制蓝色通道，将其它的眼睛关闭只显示刚刚复制好的蓝 拷贝通道

2、在蓝 拷贝通道点击选择：滤镜 >其他 >高反差保留（半径：8像素），确定后，接着使用画笔 [B画笔工具]（颜色：#9f9f9f 大小：10像素，硬度：100%，模式：正常，不透明度：100%，流量100%），涂抹人物的眼睛和嘴巴

3、在蓝 拷贝通道点击选择：图像 >计算（混合：强光），确定后得到Alpha 1，接着再重复步骤两次得到Alpha 2、Alpha 3即可

4、按着Ctrl快捷键并同时点击Alpha 3的通道缩略图，这个时候选中的是人物身体上的光滑部分，接着选择反选 [Ctrl+Shift+I 反向选择]

5、之后进入背景图层区，点击人物的背景图层，点击选择调整下面的：曲线，就可添加一个曲线图层，拖拽点线调整到人物皮肤光滑为止

6、选择盖印可见图层 [Ctrl+Alt+Shift+E盖印可见图层]，接着将背景图层复制两个[Ctrl+J复制拷贝图层]，分别命名为“表面模糊”图层、“红色通道高反差保留”图层

7、对“表面模糊”图层点击选择：滤镜 >模糊 >表面模糊（半径：20像素，阈值：25色阶）

8、对“红色通道高反差保留”图层点击选择：图像 >应用图像（通道：红，混合：正常）；接着点击选择：滤镜 >其他 >高反差保留（半径：0.8像素）；还要在混合更改{正常}栏框下点击选择线性光

9、点击选择创建新组，除了背景图层，将其它的图层都拖进新建组的组1里；接着点击选择组1、再点击添加图层蒙版，选择画笔 [B画笔工具]（大小：20像素，硬度：0%，模式：正常，不透明度：80%，流量100%），用白颜色将人物脸上的雀斑、痘印、肤质不好的地方涂抹擦除即可

注：选择点击添加图层蒙版后，画笔的颜色设置（使用选黑色是要除去的部分，使用选白色是要显示出来的部分）

10、选择盖印可见图层 [Ctrl+Alt+Shift+E盖印可见图层]，接着新建一个图层 [Shift+Ctrl+N新建图层]（颜色：灰色，模式：柔光，把“填充柔光中性色50%灰”打勾）确定即可

11、使用画笔 [B画笔工具]（颜色：#ff00ff 大小：10像素，硬度：0%，模式：正常，不透明度：35%，流量100%）涂抹在嘴唇上当做口红；也可以使用不同的颜色涂抹在眼皮上当做眼影等……

12、点击选择调整下面的：色彩平衡（色调：中间调，青色和红色+50、洋红和绿色+20、黄色和蓝色+10），这样可以调整画风的整体色调

13、最后，合并 [Ctrl+Shift+E合并可见图层]，保存[Shift+Ctrl+S保存当前图像]

制作GIF图的10个注意事项；2018.12.26

GIF(Graphics Interchange Format)的原义是“图像互换格式”。GIF文件的数据，是一种基于LZW算法的连续色调的无损压缩格式；其压缩率一般在50%左右，它不属于任何应用程序。GIF格式可以存多幅彩色图像，如果把存于一个文件中的多幅图像数据逐幅读出并显示到屏幕上，就可构成一种最简单的动画！

1、【不要提供多余无效的内容】

虽然设计师会在自己所热爱的项目中投入更多，但是在做Gif动态图的时候，在传达信息的同时，要尽量简化其中的内容，一般而言，更少的信息能够让你在制作动画的时候获得更多的自由和可能性。

一定要克制住加入大量内容的冲动；用户在观看你的Gif图的时候，很容易被过多的内容所淹没，尤其是当你想要借助Gif图来传达特定的情绪和感受的时候，多余的色彩和内容很容易弱化它们。

不要过度设计；围绕着核心的元素和调色板来设计，让它们足够独特和诱人。

2、【在Gif图中建立一致的视觉】

Gif图是世界上最短的电影，你可以借助它来进行品牌化的传达。

你可以在设计的时候使用品牌的配色来对它进行设计，将品牌的形象、Logo和其他元素在Gif图中呈现，让品牌、企业和产品以更加富有活力的方式呈现出来；而用户则能够借助这些元素，来和品牌本身产生关联。在特定的情形和需求之下，Gif图能够帮助用户了解产品的功能和其他的信息，而品牌化的配色则能够强化这种关联。

比如GE就在Gif图当中提供了非常值得期待的氛围，在官方的Tumblr当中分享这些有趣的Gif图，展示GE在科学和机械设计上的造诣。借助#badassmachines 这样的标签，让他们在社交媒体上进行更有效的传播（如下列图）：

3、【删除Gif图中所有的额外帧】

当你在设计Gif图的时候，无论你使用哪种类型的动画，Gif图中总会有某些时刻是状态停止的，创建好Gif图之后，应该通过调整，删除掉额外的、停止不同的帧。

删除额外的帧，仅保留其中一帧，并根据动画的状态和节奏来调整该帧的时长，可以缩减Gif图的大小和播放时候的压力。

这样的调整不仅能够让 Gif图整体质量保持不变，而且同时可以减少不必要的空间占用。

4、【在上传Gif图的时候，保持良好的可访问性】

作为设计师，就应该熟悉用户群体，并且将 Gif图中可能会分散精力的视觉内容给去掉。最典型的，如果你所面对的用户群体当中有视力障碍并且使用屏幕阅读功能或者癫痫患者，那么尽量兼顾到他们的需求，文字内容尽量放在Gif之外，单独显示，并且避免频繁的闪烁效果。

不断移动的Gif图会很大程度上转移用户的注意力，在一屏当中使用多个Gif图的时候，尤其要注意这一点。如果有大量文本内容的时候，Gif图中的色彩和特效尽量控制数量，因为它们可能会过多抓取用户注意力（如下列图）：


避免 Gif图自动播放，这样让用户感到可控，并且可以节省流量

5、【借助故事板来梳理Gif图的内容】

很多时候，Gif图本身的简单形态使得它看起来不复杂，但是这其实使得很多设计师因此而出错。通常，Gif图本身迷人之处就是在于它的故事性，即使再短，它当中也包含着叙事，故事性，甚至复杂的转折和多变的转场。

基本的故事变化和元素，这些是Gif图制作的基础；在设计动画的时候，我们很容易忘记这些东西，借助故事板，可以帮你更好地梳理故事的走向，镜头的变化，情节的走向，值得注意的关键情节和转折（如下列图）：


尤其是当你不是一个人来制作Gif图的时候，整个团队可以借助故事板，将进度统一起来，极大的改善沟通和统一的问题，加快创作过程

6、【如果使用PS来制作Gif图，请务必保持整洁简练】

在设计Gif图的时候，很多设计师会选择使用PS来制作。一方面是足够习惯，足够顺手，但是另外一方面，PS 本身的图层管理和制作机制还是很容易导致混乱的。

PS中元素和图层管理需要兼顾的太多，以至于在制作Gif图的时候，稍微增删或者移动一些内容，就很容易输出一个不理想、不可控的Gif图。

同时，如果没有正确地标注图层，之后又进行了相对复杂的操作之后，很容易导致多米诺骨牌一样的连锁反应错误；更令人头疼的是，修改一个地方之后，后续很多图层都需要进行调整和修改，最终会带来大量的重复性的工作。

所以，在PS中制作Gif图，要尽量保持整洁，条理清晰。

7、【增加创意元素，并保持循环】

Gif图在绝大多数情况下会自动循环播放，而用户很多时候也乐于长时间的循环观看，但是想要让它平滑自然地循环并非易事。

如果你想让 Gif图中保持循环，那么建议设计制作的时候尽量从一个简单的设计元素入手，比如几何形状之类的东西；这样可以更容易进行首尾位置的匹配（如下列图）：


在Gif图当中创建循环很大程度是希望留住用户，因为任何不协调和错位都会让人注意到，不够完美不够平滑，就会让用户感到跳出感，从而移开注意力。而这种要求，也意味着设计师要注意每一帧的设计；当然，努力的最终结果是值得的

8、【运动状态要保持模糊效果】

Gif图当中的模糊效果，其实和老式相机中的运动抓拍效果是一致的，很多处于中间态的运动，其实是不那么清晰的。而这一点和我们的视觉是保持一致的。如今相机的成像质量越来越高，但是我们的眼睛并没有随之快速的进化。

在Gif图当中，帧数通常不会太高，如果图片的细节始终保持清晰则容易呈现出跳帧，运动状态的物体适当保持模糊效果，则更接近我们日常观察的细节，从而让Gif图带来接近视频拍摄的效果，用户也会因此感到真实和精彩（如下列图）：

9、【Gif图要尽可能小】

社交媒体、博客、网站、电子邮件，甚至APP都是Gif使用大户，Gif图片已经深入到我们生活的方方面面。但是不同地方的图片规格不同，使用场景也不一样，因此，Gif需要足够小才能兼容不同的需求。（以下是一些可以缩小Gif图尺寸的方法）：

9.1尽量让 Gif图的帧数保持在150帧以内，否则在很多地方都会被限制，因为太大了。

9.2导出 Gif图的时候，请将有损压缩设置在5~10之间，这样将更大程度的压缩最终的图片尺寸。你可以多加尝试，尽量寻找质量和尺寸上的平衡点。

9.3使用尽量少的颜色。这不仅仅是出于设计的考虑，更少的色彩能够很大程度上降低文件的大小，意味着 Gif图可以更小，或者持续时间更长，加载更快。

9.4尽量避免渐变。透明度是开启还是关闭会直接影响到 Gif图的大小，如果想要确保文件不那么大，尽量避免吧。

10、【针对移动端设备进行设计】

这个注意事项其实设计制作的时候也应该早有预料。现如今的用户绝大多数的时间都消耗在移动端的小屏幕上了。

在设计的时候，注意使用更小的视觉元素，避免智能手机用户信息过载；另一方面，如果某些元素需要更加吸引用户的注意力，可以使用更加大胆的颜色，或者借助动画效果，都是可以的。

同时，考虑到社交媒体上图片素材的展现形式，可以尽量把 Gif图设计成为方形的，这样可以更好的兼容不同的屏幕和不同的平台，同时也方便裁切（如下列图）：
